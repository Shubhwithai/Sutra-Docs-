## ðŸš€ Get Started

### Prerequisites
- **Python**: v3.8 or higher
- **Required Packages**: 
  ```bash
  pip install streamlit langchain-openai python-dotenv PyPDF2 faiss-cpu langchain-community
  ```
- **API Key**: Obtain your SUTRA API key from [https://www.two.ai/sutra/api](https://www.two.ai/sutra/api)
- **Environment Setup**: Create a `.env` file with `SUTRA_API_KEY=your_key_here`

### Running the Apps
1. Clone the repository: `git clone https://github.com/sutra-dev/sutra-cookbook.git`
2. Navigate to the desired app directory: `cd sutra-cookbook/starter-apps/[app_directory]`
3. Run the app: `streamlit run app.py`
4. Access in your browser at `http://localhost:8501`
