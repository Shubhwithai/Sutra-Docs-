## Supported Integrations

- **LangChain**: Build advanced chains and workflows using SUTRA as a language model backend.
- **LlamaIndex**: Use SUTRA for retrieval-augmented generation and document QA pipelines.
- **Portkey**: Connect SUTRA to Portkey for secure, scalable API management.
- **LiteLLM**: Integrate SUTRA with LiteLLM for lightweight, multi-provider LLM orchestration.
- **CrewAI**: Power autonomous agents and task automation with SUTRA.
- **Agno**: Use SUTRA in Agno pipelines for advanced reasoning and analytics.

## Example: Using SUTRA with LangChain (Python)

```python
from langchain.llms import OpenAI

llm = OpenAI(
    base_url="https://api.two.ai/v2",
    api_key="YOUR_SUTRA_API_KEY"
)

response = llm("Translate this to Hindi: 'How are you?'")
print(response)
```
